  1. Using the symbol on your web pages

    To use the symbol on your web pages, include the following stylesheet <link rel="stylesheet" href="http://axemclion.github.com/rupee/rupee.css"/> Then, add the following code at places where you would like the R to show up. <span style ="font-family:rupee">R</span> For example, to say that something costs R 100, write <div>something costs <span style ="font-family:rupee">R</span> 100 <div> This page uses the rupee symbol font. You can select it like normal text; it is not an image. More information on the details at http://blog.nparashuram.com/search/labels/rupeesymbol
